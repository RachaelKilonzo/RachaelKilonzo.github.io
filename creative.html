Creative Coding & Generative Art Gallery

Projects exploring the intersection of data, algorithms, and art, primarily built using Node.js and client-side rendering libraries.

Project 1: Data-Driven Generative Waveforms

A Node.js backend processes real-time weather data and feeds it to a p5.js frontend sketch, which generates dynamic, abstract visualizations based on temperature and wind speed. Demonstrates API integration and front-end rendering.

  • View Live Art Demo
  • Source Code (Node/p5)

Project 2: 3D WebGL Data Scatter Plot

Used a Node.js server to prepare a large, cleaned dataset and visualize it in 3D space using Three.js. This piece functions as both an artistic installation and an interactive data exploration tool.

  • View 3D Visualization
  • View Node Backend Code

Project 3: Procedural Content Twitter Bot

A Node.js script that runs daily, fetching trending data and using a set of procedural rules to generate a small visualization (SVG) which is then automatically posted to Twitter. Focus on automation and scheduling.
